# Video Search Platform

## Requirements

* npm
* [morton/vds3](http://git-cn.viscovery.co/morton/vds3)

## Setup

``` bash
    $ npm install
    $ npm run develop
```

## Usage

After setup, access http://localhost:8080/ for usage.

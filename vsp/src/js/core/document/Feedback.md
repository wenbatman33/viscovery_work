# Feedback

## Banner

## Props
| Name | Type | Default |
| ------ | ----------- | --- |
| vdStyle | `bool` | false |

### Example
```js
import { Banner } from 'vidya/Feedback';

<Banner>
  我的朋友，逃往你的孤獨裡去，逃往吹刮著強烈的暴風的地方去吧。你的命運不是叫你做蒼蠅拍子。
</Banner>
```

## Bubble
### Example
```js
import { Bubble } from 'vidya/Feedback';

<Bubble>
  同一個軀殼裡，住著兩種情感的肉體，強與弱對峙，熱與冷相逼
  ，當有一方耗盡力氣時，另一方反撲回來接手殘局。
</Bubble>
```

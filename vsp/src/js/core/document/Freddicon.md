# Freddicon

## FAIcon

## Usuage
for font-awesome icon, directely add icon name to 'faClassName' (like: fa-calendar)
check font-awesome icon list:
http://fontawesome.io/icons/

## Props
| Name | Type | Default |
| ------ | ----------- | --- |
| faClassName | `String` |  |
| className | `String` |  |
| style | `Object` |  |
| onClick | `Function` |  |

### Example
```js
import { FAIcon } from 'vidya/Freddicons/FAIcon';

<FAIcon
  className={styles.icon}
  faClassName="fa-calendar"
/>
```
